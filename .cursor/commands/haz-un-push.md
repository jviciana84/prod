# Haz un push

Sube todos los cambios del proyecto a Git (add, commit, push) para que el dominio se actualice desde el repositorio.

- Ejecutar en: `C:\Users\Usuario\Downloads\C18` (proyecto C18 portfolio; cursor tiene su propio Git y se sube aparte)
- Comandos: `git add .` → `git commit -m "mensaje"` → `git push origin main`
- Pedir permiso de red para el push.
