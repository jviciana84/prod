## 📏 Dimensiones y Espaciado

- **Formato:** A4 (210mm x 297mm)
- **Márgenes:** 40pt (todos los lados)
- **Padding interno:** 15-20pt
- **Separación entre secciones:** 20pt
- **Separación entre elementos:** 8-15pt

## 🖼️ Imágenes

- **Documentos (Pág 1):** 48% width, 200pt height
- **Fotos (Pág 3-N):** 48% width, 180pt height
- **Borde:** 1pt solid #e5e7eb
- **Border radius:** 4pt

- **Datos textuales:** 15%
- **Imágenes:** 80%
